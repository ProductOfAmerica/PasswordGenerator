Privacy Policy

Last updated: March 24, 2026

Overview

Strong Password Generator is designed with privacy as a core principle. The extension operates entirely offline and does not collect, transmit, or sell any personal data.

Data Collection

This extension does not collect any personally identifiable information. No analytics, telemetry, or tracking of any kind is used.

Password Generation

All passwords and passphrases are generated locally in your browser using cryptographically secure randomness. Generated passwords are never stored, logged, or transmitted. They exist only in memory while the popup is open.

Settings Storage

Your preferences (generation mode, character count, toggle states, word count, and separator choice) are saved using Chrome's built-in storage API (chrome.storage.sync). If you are signed into Chrome, these settings may sync across your devices via your Google account. No passwords are ever stored — only configuration preferences.

Clipboard

When you click a generated password to copy it, the extension writes it to your system clipboard. The clipboard is automatically cleared after 30 seconds for security.

Network Requests

This extension makes zero network requests. All assets, including fonts, are bundled locally. The extension functions fully offline.

Third Parties

No data is shared with any third party. The extension has no external dependencies at runtime.

Permissions

The only permission requested is storage, used solely to persist your configuration preferences across sessions.

Chrome Web Store Limited Use Compliance

This extension's use of chrome.storage.sync complies with the Chrome Web Store Limited Use requirements. User data is used solely to provide and improve the extension's core functionality (persisting generation preferences). Data is not transferred to third parties, not used for advertising, and not used for creditworthiness or lending purposes.

Contact

If you have questions about this privacy policy, please open an issue at github.com/ProductOfAmerica/PasswordGenerator.
